# ROADMAP
### A1 Game Academy

## Princípio

Cada projeto da Academia é escolhido para ensinar Python através de um jogo,
mas o objetivo vai além do jogo em si: as habilidades desenvolvidas devem
ser transferíveis para outras áreas — aplicativos, automação, banco de
dados, ciência de dados, IA generativa, entre outras.

O jogo é o veículo. O aprendizado é o destino.

**Foco atual: um jogo de cada vez, levado até a maturidade completa.**
A Academia não abre um segundo projeto apenas para diversificar — um novo
jogo só começa quando o atual atingir um patamar de conclusão que justifique
partir para o próximo desafio técnico.

## Fase 1 — Forgotten Ship
**Status:** Em andamento

Primeiro e único projeto ativo da Academia no momento. Seu roadmap interno
(fases do próprio jogo) é documentado separadamente, no repositório do
projeto: `projetos/001-forgotten-ship/docs/ROADMAP.md`.

A Fase 1 da Academia permanece aberta enquanto o Forgotten Ship continuar
sendo o veículo principal de aprendizado — o que pode incluir expansões
grandes de escopo dentro do próprio jogo.

## Fase 2 — Segundo Projeto
**Status:** Não iniciada — sem previsão

Só será definida quando a Fase 1 atingir maturidade suficiente. Ainda não
há decisão sobre gênero, tecnologias-alvo ou objetivo de aprendizado do
segundo projeto.

## Fase 3 — Além dos Jogos
**Status:** Visão de longo prazo

Transição do aprendizado de jogos para aplicações reais: aplicativos,
ciência de dados, automação, IA generativa. Jogos deixam de ser o único
veículo de ensino da Academia.

## Histórico de Versões

### v3.0
- Simplificado para refletir o foco em um único projeto por vez, até
  maturidade completa, em vez de planejar múltiplos jogos em paralelo.
- Roadmap interno de cada jogo passa a viver no repositório do próprio
  projeto, não mais aqui.

### v2.0
- Reescrito para expressar o princípio de progressão de aprendizado
  (jogos como veículo para habilidades transferíveis), não apenas
  uma lista de projetos.

### v1.0
- Versão original, criada antes do início do desenvolvimento.
